# Thực hành FlatList trong React Native

## Thông tin sinh viên
- **Họ và tên:** Nguyễn Thị Huệ Minh
- **Mã sinh viên:** 23810310177
- **Môn học:** Lập trình React Native
- **Bài thực hành:** Sử dụng FlatList hiển thị danh sách thông báo

## Mô tả bài thực hành
Ứng dụng React Native sử dụng FlatList để hiển thị danh sách thông báo/công việc.
Mỗi phần tử trong danh sách bao gồm tiêu đề, mô tả và thời gian, mô phỏng màn hình thông báo trong ứng dụng di động.

## Nội dung thực hiện
- Sử dụng FlatList để hiển thị danh sách
- Tối ưu hiệu năng so với ScrollView
- Thiết kế giao diện item thông báo
- Quản lý dữ liệu bằng mảng object

## Công nghệ sử dụng
- React Native (Expo)
- JavaScript

## Cách chạy ứng dụng
